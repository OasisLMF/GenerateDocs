generate_oasis_structures(self)[source]¶

Create the Oasis structures - FM Programmes, FM Profiles and FM Policy TCs - that represent the reinsurance structure.

The algorithm to create the stucture has three steps: Step 1 - Build a tree representation of the insurance program, depending on the reinsurance risk level. Step 2 - Overlay the reinsurance structure. Each reinsurance contact is a seperate layer. Step 3 - Iterate over the tree and write out the Oasis structure.
